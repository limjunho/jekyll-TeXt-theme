### Solution

**재귀 함수란 어떤 함수에서 자신을 다시 호출하여 작업을 수행하는 방식의 함수를 의미한다.**  

### Code  
```cpp
#include <iostream>

using namespace std;

int fectorial(int num)
{
    if (num > 1)
        return num * fectorial(num - 1);
    else
        return 1;
}

int main()
{
    int N;

    cin >> N;

    printf("%d\n", fectorial(N));

    return 0;
}
```
