**git은 파일명 또는 폴더명의 대소문자를 구분하지 못한다.**  
따라서 git 설정에서 대소문자를 무시하지 않도록 설정이 필요하다.

```bash
git config core.ignorecase false
```

이제 파일명을 변경하여도 git이 잘 추적할 것이다.
