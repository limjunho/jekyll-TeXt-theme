## nohup

& 명령을 이용해 백그라운드로 실행하더라도 터미널을 종료하면 프로세스도 종료되게 된다.  
예를 들어 서버를 실행시키고 터미널을 유지하지 않더라도 서버 프로세스를 유지하고 싶다면 & 명령만으로는 어렵다.  
아래 명령을 활용하면 터미널 세션을 끊더라도 프로세스가 종료되지 않는다.  

```bash
$ nohup node /절대 경로/index.js &
```

**node server를 실행시키는 예**  

**프로세스 확인**  
```bash
$ ps -aux | grep 프로세스명
```

**프로세스 종료**  
```bash
$ kill -9 PID
```
