### 데이터베이스 생성  

```bash
mysql> show databases;
```
**데이터베이스 조회**  

```bash
mysql> create database databasename;
```
**데이터베이스 생성**  

### 데이터베이스 삭제  

```bash
mysql> show databases;
```
**데이터베이스 조회**   

```bash
mysql> drop database databasename;
```
**데이터베이스 삭제**  

### 데이터베이스 권한부여  

```bash
mysql> grant all privileges on databasename.* to 'username'@'localhost or %';
```
**권한 부여**  

```bash
mysql> show grants for username@'localhost or %';
```
**권한 조회**
